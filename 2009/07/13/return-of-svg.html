Raphaël. The Return of Browser SVG

Jul 13 2009

Maybe we've all been too hasty in declaring the canvas tag the new king of web interaction. I've just stumbled upon Raphaël—a javascript library for interacting with SVG. The library is cross-browser thanks to VML on Internet Explorer.

I really like the idea of having real DOM nodes when manipulating things via javascript. The canvas tag always seemed a big too much like Flash for my liking. Here are a few of my favorite demos:

  • A vector map of Australia
  • Github-style impact chart
  • Interactive analytics graph
  • Draggable graph

Thanks to Josh Pyles for pointing it out.
